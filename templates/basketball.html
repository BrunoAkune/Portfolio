The Problem

Sports fans and analysts often struggle to quickly access and visualize NBA team performance data in a meaningful way. Raw statistics from multiple games need to be aggregated, processed, and presented in an interactive format that allows for easy comparison and trend identification. The challenge was to build a solution that could fetch real-time data from an external API and transform it into actionable insights.

My Role

Designed and developed a full-stack web application from scratch, handling every aspect of the project:

  • Architected the backend using FastAPI with WSGI middleware integration for Dash
  • Implemented a caching system with 5-minute TTL to handle API rate limits gracefully
  • Built interactive Plotly visualizations showing score trends and win/loss differentials
  • Created data normalization logic to handle home/away game statistics
  • Designed the dashboard UI with Lakers-branded color scheme

The Impact

  • Created a real-time dashboard displaying Lakers game statistics including win/loss records, scoring trends, and point differentials
  • Built a resilient caching system that prevents dashboard failures during API rate limiting (429 errors)
  • Demonstrated proficiency in API integration, data transformation, and interactive visualization
  • Showcased ability to build production-ready applications with proper error handling and fallback mechanisms
